# 1차 오픈(소프트런칭) 기능 스코프

> 기준: 소울헌터 키우기 **출시 시점(v1, 2025-12-17)** 기능 세트 역산 + BM 기획서 P0.
> 원칙: **"1주 분량 콘텐츠가 준비되면 출시"** (룡게임 원칙) — 완벽이 아니라 격주 업데이트를 버틸 최소 골격.
> 소헌키도 출시 때는 길드(v2)·전직(v3)·아레나(v3)가 없었다. 코어 루프 + BM + 일일 루틴이면 연다.

## ✅ 완료 (현재 상태)

| 영역 | 기능 |
|---|---|
| 코어 루프 | 방치 전투(수식 기반), 스테이지 등반, 벽, 오프라인 보상(상한 8h/멤버십 12h) |
| 성장 | 성장 축 5종(수련×2/영혼 주입/명계 계약/달빛 각성), 장비 개별 강화 200레벨(등급별 비용) |
| 소환 | 배너 5종(낫/오브/장식/스킬/차사), **소환 레벨 Lv1~10 실측 확률표**, 확률 공시, 등급 6단계 |
| 수집 | 장착 슬롯제(1/1/2/4/4), 돌파(10돌)+임계 효과, 도감 보유 효과+마일스톤, 합성(잉여→상위 등급) |
| 반복 | 던전 3종(골드/영혼/각성석), 일일 무료 3회+소탕권+광고 소탕 |
| BM | 구독 3종(연금/멤버십/**광고 스킵**), 페이백 출석, 패키지 사다리(90~2,990), 진행도 트리거 팩, 마일리지 상점, SKU 6종, FakeStore/FakeAd |
| UX | 튜토리얼(FTUE 6단계), 5탭 UI, 정보(세부 스탯) 탭, 저장/마이그레이션 |
| 개발 인프라 | 순수 C# 코어+테스트 51개, 밸런스 시뮬레이터(무/소과금 봇), 데이터 주도(config.json) |

## 🔲 P0 — 오픈 전 필수 (남은 것)

### 게임플레이 (약 1~2주)
1. **일일 미션 + 무료 출석부** — 일일 루틴의 뼈대. 미션 보상이 영옥 수입원(현재 시뮬의 '일일 무료 영옥 800' 가정을 실제 시스템으로)
2. **보스전 구분 연출** — 5단계마다 보스 도전 버튼/실패 표시 (현재는 자동 도전이라 벽 체감이 약함)
3. **전투 비주얼 최소치** — 몬스터 처치 카운트/타격감 연출 (텍스트→간단 스프라이트)
4. 광고 슬롯 UI 연결 마무리 — 오프라인 팝업에 [광고 보고 2배] 버튼, 무료 일일 팩/무료 소환 슬롯 노출

### BM 연동 (약 1~2주)
5. **Unity IAP 연동** (구글 우선) + 스토어 콘솔 상품 등록 — IStoreAdapter 구현체만 추가
6. **AppLovin MAX(또는 AdMob) 연동** — IAdAdapter 구현체
7. **첫 충전 2배** (SKU별 1회)
8. 이벤트 템플릿 1세트 (출석 2트랙+미션+스텝업+교환소 — 오픈 기념 이벤트로 가동)

### 아트/스토어 (외주·AI 병행, 약 2주)
9. 유닛 일러스트 30종 + 저승사자 캐릭터 + 배경 1장 (AI 생성 → 선별)
10. **앱 아이콘 + 스토어 스크린샷 5장** — CPI를 결정하는 유일한 아트, 여기만 품질 투자
11. 폰트 교체(무료 한글 폰트) + 사운드 최소치(BGM 1, 타격음)

### 인프라 (약 1주)
12. **클라우드 세이브 1개** — Google Play Games 연동(최소) 또는 뒤끝. 기기 분실 = 계정 분실이면 과금 유저가 못 생긴다
13. Android 빌드 파이프라인 + 자체등급분류 심의
14. 개인정보처리방침/이용약관 페이지 (스토어 필수)

## 🚫 오픈에 넣지 않는 것 (소헌키 검증: 없어도 출시 가능)

- 길드 → **v2 (오픈 +2주)** 업데이트 카드
- 전직/직업 시스템 → v3
- PvP(아레나) → v3~4
- 고대 등급 → v6± (등급 인플레 카드는 아낀다)
- 시즌제 콘텐츠 → v11±
- 쿠폰 시스템(서버 필요) → 뒤끝 붙일 때 함께

## 오픈 판정 기준 (시뮬레이터 + CBT)

- [ ] 무과금 봇 D1 8~12장 / D7 20장± / D30 30장± (상한 60장의 절반)
- [ ] 소과금 봇이 무과금 대비 D30 기준 +2~4장 우위
- [ ] 첫 벽이 D2~3에 오고, 벽 시점에 트리거 패키지가 노출됨
- [ ] 튜토리얼 완주 5분 이내
- [ ] CBT(지인 10~20명) 1주 — 소헌키도 출시 1주 전 CBT
- [ ] D1 리텐션 25%+ 나오면 오픈, 미만이면 초반 훅 보강 후 재측정

**예상 총 소요: 4~6주** (아트 외주 병행 시). 오픈 후엔 격주 업데이트 컨베이어(GM노트+쿠폰+신규 축 1개)로 전환.
