# 저승사자 키우기: 방치형 — BM 기획서 v1.0

> 기준: 소울헌터 키우기 실측 분석(`research/soul-hunter-idle-analysis.md`) + 2026 시장 벤치마크.
> 원칙: **"과금 천장은 높게, 소과금 정착 경로는 확실하게, 첫 결제는 공짜처럼."**
> 모든 수치는 시뮬레이터(`tools/simulator`)로 검증 가능해야 하며, 이 문서가 곧 `config.json`의 명세다.

## 1. 재화 체계

```
₩ (실결제)
 └→ 수정 (hard, 유일한 원화 게이트) ..... 1수정 ≈ ₩10
      └→ 상품/패스/픽업 리셋/광고 스킵
 영옥 (soft, 보상 재화) ................. 소환(가챠)의 주 재화
 골드 / 영혼 ............................ 파밍 성장 재화 (수련/주입 축)
 각성석 / 명부 조각 .................... 후반 병목 재화 (던전 전용)
 소탕권 (던전별) ........................ 시간 단축권 = 쿠폰/패키지의 겨냥점
```

- **원화 가격은 게임 내 어디에도 표기하지 않는다.** 모든 상품은 수정 표기 (소울헌터 검증 패턴).
- 수정 가격 사다리 고정: **90 / 190 / 290 / 490 / 990 / 1,490 / 1,990 / 2,990 / 4,900**
- 영옥과 수정의 교환은 없음 (수정은 오직 상품 구매용 → 가치 희석 방지).

### 1-1. 수정 충전 SKU (스토어 등록 상품)

| SKU | 가격 | 기본 수정 | 보너스 | 첫 충전 2배 시 |
|---|---|---|---|---|
| gem_1100 | ₩1,100 | 110 | — | 220 |
| gem_5500 | ₩5,500 | 550 | — | 1,100 |
| gem_11000 | ₩11,000 | 1,150 | +5% | 2,300 |
| gem_33000 | ₩33,000 | 3,650 | +11% | 7,300 |
| gem_55000 | ₩55,000 | 6,300 | +15% | 12,600 |
| gem_110000 | ₩110,000 | 13,200 | +20% | 26,400 |

- **첫 충전 2배**는 SKU별 1회 (전 SKU 순회 유도 — 총 ₩215,600어치 2배 혜택이 고래 온보딩).
- ₩1,100 SKU는 "결제 심리 문턱 제거"용 — 페이백 출석(490수정)이 첫 충전 2배(220수정)로는 부족하게 설계되어 자연스럽게 ₩5,500으로 유도.

## 2. 상품 카탈로그 (전체)

### 2-1. 구독형 — 기반 매출 (핵심 중의 핵심)

| 상품 | 가격 | 내용 | 설계 의도 |
|---|---|---|---|
| **윤회 연금** | 1,490수정 / 30일 | 매일 수정 100 지급 (총 3,000 = 2.0배 환급) | 소울헌터 연금 복제. 매일 접속 이유 + 수정의 정기 공급원 |
| **명부 계약** (VIP 멤버십) | 990수정 / 30일 | ① 오프라인 상한 8h→12h ② 오프라인 보상 +50% ③ **월혼(月魂) 성장 축 해금** ④ 던전 소탕 일 +2회 | 소울헌터 '달의 가호 잠금' 패턴 — 미구매 시 성장 축 하나가 사실상 잠김. 방치형의 정체성(방치 효율)을 정면으로 판다 |
| **광고 패스** | 990수정 / 30일 | 보상형 광고를 **보지 않고** 보상 수령 (Skip'it 패턴) | 광고 수익을 IAP로 전환. Mob Control 사례: IAP 매출의 18%+ |

→ **소과금 정착 세트 = 연금 + 명부 계약 = 2,480수정 ≈ ₩24,800/첫 달, 이후 월 ₩15,400** (첫 충전 2배 활용 시).
커뮤니티 공략이 "월 ₩15,400으로 평생"이라고 쓰게 만드는 것이 목표 (소울헌터의 평판 전략).

### 2-2. 첫 결제 전환 장치

| 상품 | 가격 | 내용 | 설계 의도 |
|---|---|---|---|
| **환생 출석부** (페이백 출석) | 490수정 | 구매일부터 7일 출석 → 매일 영옥 3,000 → **완주 시 490수정 전액 환급** | "공짜나 다름없는 첫 결제." 구매일 카운트 시작 = 7일 리텐션 장치를 겸함 |
| **첫걸음 패키지** | 490수정 · 계정당 1회 | 영옥 20,000 + 골드 대량 + 신화 차사 선택권 1 | 가치 1,200% 표기. 페이백 완주 환급 수정의 사용처 = 결제 루프 완성 |
| 첫 충전 2배 | — | SKU별 1회 | 표준 |

**퍼널**: 무료 광고 팩 체험 → 페이백 출석(₩5,500 첫 충전) → 환급 490으로 첫걸음 패키지 → 소과금 세트 안내 → 정착.

## 3. 게임 밸런스 × BM 결합 설계

### 3-1. 성장 축 로드맵 (곱연산 축 = 격주 매출 공급)

| 축 | 해금 | 재화 | 곱연산 기여 | BM 접점 |
|---|---|---|---|---|
| 수련 (공격/신속) | 시작 | 골드 | 가산 기반 | 골드 폭탄 패키지 |
| 영혼 주입 | 시작 | 영혼 | ×FinalDamage | 일일/주간 팩 |
| 명계 계약 | 5장 | 영혼 | ×GoldGain | — |
| **월혼(月魂)** | 12장 | 월혼석 (**방치 보상 전용**) | ×FinalDamage | **명부 계약 멤버십이 획득 2배** ← 잠금 축 |
| 달빛 각성 | 12장 | 각성석 (던전) | ×FinalDamage | 소탕권 경제 |
| 장비 강화/초월 | v0.3 | 골드+강화석 | ×부위별 | 후반 골드 싱크 (시뮬 확인된 골드 잉여 해소) |
| 유물/도감 | v0.4 | 수집 | ×소량 다수 | 가챠 |

**격주 업데이트 = 이 표에 행 하나 추가** (신규 축 + 전용 던전 + 소탕권 패키지 3종 세트).

### 3-2. 벽 스케줄 (의도된 정체 = 과금 또는 방치 선택지)

```
D1  : 벽 없음 — 10분에 한 번씩 파워스파이크 (전직·첫 10연 무료·신규 축 해금)
D2-3: 첫 벽 (10장 보스) — "하루 방치하면 뚫림" 체감 → 방치형 문법 학습 + 990 패키지 첫 노출
D6-8: 영혼 병목 (20장) — 주간 팩/멤버십 가치 극대화 시점 = 구독 안내 타이밍
D14+: 각성석 병목 (30장) — 던전 소탕권 경제 본격화
D30+: 시즌 콘텐츠·아레나가 성장 외 목표 제공 (성장 정체를 경쟁/수집으로 상쇄)
```

**검증 방법**: `tools/simulator` 봇의 정체 감지(⚠ 벽)가 위 스케줄과 일치할 때까지 config.json 튜닝. 무과금 D30 도달 스테이지는 콘텐츠 상한의 50~60%를 유지 (버섯커·소울헌터 공통 패턴 — "무과금도 끝까지는 감").

### 3-3. 오프라인 보상 설계 (방치형의 심장)

- 기본 상한 8h = **수면 시간과 정확히 일치** → 아침 수령이 최적 행동 = 일일 첫 접속 고정.
- 멤버십 12h → 퇴근 후 수령까지 커버 = 직장인 타겟.
- 오프라인 효율 60% (온라인 대비) → "접속하면 더 좋다"의 여지 = 세션 유도.
- **광고로 오프라인 보상 2배** (일 3회) → 무과금의 아침 루틴에 광고 3회 고정.

### 3-4. 소탕권 경제

던전(골드/영혼/각성석/유물) 일 3회 무료 + 광고 +2회 + 멤버십 +2회.
소탕권은 쿠폰·패키지·패스 보상의 단골 (소울헌터: 쿠폰 보상이 병목 재화 겨냥).
**소탕권의 실질 가치 = 시간** → 판매하는 것은 아이템이 아니라 시간 단축.

## 4. 킬러 컨텐츠 (후킹 장치)

### 4-1. 초반 후킹 (D0 — 이탈 방어)
- 설치→전투까지 30초 (튜토리얼 스킵 가능, 낫질 한 번이 첫 입력).
- 10분 내: 첫 전직급 연출 + 무료 10연 소환 + 오프라인 보상 미리보기("잠들면 이만큼 쌓입니다").
- 첫 세션 종료 유도: "지금 끄면 8시간 뒤 ○○골드" — **끄는 것도 전략**임을 학습.

### 4-2. 수집 — 명부(名簿) 도감
- 차사·귀문술·유물 전 수집물이 도감 등록 → **보유만으로 영구 스탯** ('잉여 제로').
- 돌파 임계(신화 5돌, 고대 9돌)에 큰 효과 → 픽업 집중 과금 유도.
- 도감 완성률 마일스톤 보상 (컴플리셔니스트 후킹).

### 4-3. 경쟁 — 저승 랭킹 (v0.4)
- **아레나**: 10초 딜 측정 비동기 PvP, 주간 시즌, 티어 보상 + 1~10위 전용 외형(과시).
- **혼각 쟁탈** (이계 패턴): 영혼 광맥 점령 → 타 유저 약탈 가능. "과금하고 싶어지는" 콘텐츠. 익명 옵션으로 스트레스 조절 (소울헌터 v10 학습).
- 랭킹 상위 빌드 열람 = 공략 배포 채널 겸용.

### 4-4. 시즌 — 심연 사이클 (v0.5)
- 2주 시즌: 시즌 전용 등반 + 시즌 상점 + 시즌 패스 연동. 시즌 리셋 = 복귀 명분 상시 생성.

### 4-5. 일일 루틴 스택 (체크리스트가 곧 리텐션)
아침: 오프라인 수령(광고 2배) → 출석 → 일일 팩 → 던전 소탕 3회 → 미니게임 랭킹 확인(확인해야 보상 우편).
저녁: 오프라인 수령 → 아레나 5회 → 시즌 미션 마감.
**총 소요 10분 × 2회** — 방치형의 약속: "짧게, 그러나 매일."
