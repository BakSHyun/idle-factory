# 유닛 효과 체계 정리 (장착 / 특수 / 보유 / 돌파)

> 모든 수치는 `Assets/StreamingAssets/config.json`이 원본이며, 등급 배율(초급 1× → 신화 50×)과
> 티어 배율(1~4티어: 1.0/1.5/2.2/3.2)이 곱해진다. 이 문서는 구조 스펙.

## 등급 스케일

| 등급 | 배율 | 장비 강화비(Lv1) | 스킬 쿨타임 | 차사 스킬 쿨타임 |
|---|---|---|---|---|
| 초급 | 1× | 80골드 | 9초 | 12초 |
| 중급 | 2.2× | 200 | 8초 | 11초 |
| 상급 | 4.8× | 500 | 7초 | 10초 |
| 특급 | 10.5× | 1,300 | 6초 | 9초 |
| 전설 | 23× | 3,500 | 5초 | 8초 |
| 신화 | 50× | 9,000 | 4초 | 7초 |

## 부위별 효과 3분류

### 🗡 낫 (무기) — 슬롯 1
| 분류 | 효과 |
|---|---|
| 장착 효과 | 공격력(고정) + 무기 공격력 피해(%) · 강화(Lv1~200)마다 공격력 % 증가 |
| 보유 효과 | 공격력 증가(%) · 크리티컬 피해 |
| 돌파 비례 | 한계돌파 1개당 공격력(%) |

### 🔮 오브 — 슬롯 1
| 분류 | 효과 |
|---|---|
| 장착 효과 | 체력 + 모든 속성 피해(%) · 강화마다 모든 속성 피해 증가 |
| 보유 효과 | 골드 획득량(%) |
| 돌파 비례 | 한계돌파 1개당 공격력(%) |

### 📿 장식 — 슬롯 2
| 분류 | 효과 |
|---|---|
| 장착 효과 | 방어도 + 방어 무시 · 강화마다 방어 무시 증가 |
| 보유 효과 | 영옥 획득량(%) — 미션/출석 보상에 적용 |
| 돌파 비례 | 한계돌파 1개당 공격력(%) |

### ✨ 스킬 — 슬롯 4 (+속성 보너스)
| 분류 | 효과 |
|---|---|
| 스킬 효과 (장착) | 쿨타임형 자동 시전: 피해량(DPS×N초어치) × 타격수 |
| 스킬 특수 효과 | 스킬별 고유 1종: 공속/치확/치피/속성 피해/방어 무시/스킬 피해/최종뎀 (예: 삼도천의 심판 = 최종뎀 +8%) |
| 보유 효과 | 스킬 피해량 증가(%) |
| 돌파 마일스톤 | 1성 피해+10% → 3성 쿨감 -1초 → 6성 피해+30% → 9성 타격수+1 → **10성 최종 데미지 +100%** |

### 👤 차사 — 슬롯 4 (+속성 보너스)
| 분류 | 효과 |
|---|---|
| 스킬 효과 (장착) | 쿨타임형(N초) 또는 **확률형(평타당 18%)** 발동 · 피해량 × 타격수 · 일부는 상태이상(화상/감전/혼돈 — 속성 연동, 25% 부여) |
| 고유 패시브 (특수) | 차사마다 1종 — 예: 홍련 '홍련의 업화'(불 피해+10%), 강림도령 '강림'(방어 무시+20), 흑랑 '늑대 발톱'(타격수+1) |
| 보유 효과 | 영웅 스킬 피해량 증가(%) |
| 돌파 마일스톤 | 1성 피해+10% → 3성 쿨감 -1초(확률형은 발동확률+3%) → 6성 피해+30% → 9성 상태이상 확률+20%(무상태는 타격수+1) → **10성 공격력 +200%** |

## 관련 스탯 (신규 포함)

| 스탯 | 출처 | 적용 |
|---|---|---|
| 무기 공격력 피해% | 낫 | 공격력 곱연산 |
| 모든 속성 피해% | 오브 | 속성 보너스에 합산 |
| 방어 무시 | 장식/패시브 | 적 방어도 상쇄 (파밍·보스·생존 전부) |
| 영옥 획득량% | 장식 보유 | 미션/출석 보상 배율 |
| 스킬 피해량% (HeroSkillDamage) | 차사·스킬 보유 | 모든 액티브 스킬 피해 |
| 속성별 공격력% | 연마 축 + 패시브 | 해당 속성 파티 비중만큼 |

## 속성 시스템 요약
- 불 🔥 > 어둠 🌑 > 번개 ⚡ > 불 (상성 삼각형), 몬스터 속성 = 챕터 로테이션
- 상성 유리 유닛당 최종뎀 +6% / 불리 -4%
- **속성당 장착 슬롯 +1** (파티 속성 가짓수만큼, 최대 +3)
